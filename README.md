# Matt's Dotfiles

25+ years of dotfile tweaking has led me to this.

## Requirements

You'll need GNU Stow, `make`, and `git`.

## Installation

    git clone https://github.com/mperry2/dotfiles.git $HOME/.dotfiles
    cd $HOME/.dotfiles
    make
